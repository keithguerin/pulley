Pulley

Lightweight javascript framework.

These tools are all managed through CodeKit and the file config.codekit.

All source code is found inside of ‘html/source’. This code is processed into three primary files in ‘html’: index.html, scripts.min.js, and styles.min.js.

HTML Documents (index.html)

All HTML in this project is generated by Jade. Jade is an HTML templating engine, built on Node. It allows for a simple syntax, variables, and imports.

Styles (alt) (styles.min.js)

All styles are written in Sass. * Bourbon is a Sass mix-in library. * styles.min.css.map allows debugging and editing of styles in Chrome Developer Tools. Instructions.

Scripts (scripts.min.js)

Scripts use the MVC pattern defined by Pulley. Pulley provides a common way to organize code, and favors native JavaScript over custom syntax. * jQuery is used for selectors, event binding, and sparsely for other utilities. * scripts.min.css.map allows debugging and editing of styles in Chrome Developer Tools. Instructions.
